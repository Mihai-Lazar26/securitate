malware = Software-ul rău intenționat, software-ul dăunător este un tip de software proiectat intenționat pentru deteriorarea unui computer sau infiltrarea în el, sau/și deteriorarea ori infiltrarea în întregi rețele de computere, fără consimțământul proprietarului respectiv.

virus = Un virus este un tip e program care la executie se replica modificand programe inserandu-si propriul cod

dropper = Un dropper este un fel de troian care a fost conceput pentru a „instala” un fel de malware într-un sistem țintă. Codul malware poate fi conținut în dropper astfel încât să se evite detectarea de către scanerele de viruși sau dropperul poate descărca malware-ul pe mașina țintă odată activat.

downloader = Este un tip e troian care descarca si instaleaza programe malitioase.

trojan = Este un virus care se deghizeaza ca un program legitim.

spyware = Programele spyware sunt o categorie de software rău intenționat, atașate de obicei la programe gratuite, care captează pe ascuns date de marketing și le folosesc apoi pentru a transmite utilizatorului reclame corespunzătoare dar nesolicitate.

riskware = Riskware este un software a cărui instalare și execuție prezintă un posibil risc, dar nu definit pentru un computer gazdă.

ransomware = Ransomware este un software rău intenționat care, după ce se instalează pe dispozitivul victimei, criptează datele victimei ținându-le „ostatice” sau șantajează victima, pe care o amenință că îi va publica datele dacă aceasta nu plătește o „răscumpărare”.

adware = Adware este un program care afișează reclame de conținut promoțional la rularea acestuia, reclame care pot fi afișate ca bannere în fereastra programului, sau de tip pop-up

worm = Este un program care se replica cu scopul de a se raspani la alte computere.

obfuscare = În dezvoltarea de software, obfuscarea este actul de a crea cod sursă sau mașină care este greu de înțeles de oameni sau computere.
